一.系統簡介
這是一份軟體開發習慣普查，邀請軟體工程師進行問卷填寫，並在填寫完問卷完成後會提供一份私有的github repository作為贈品．
過 Google sheet 中的 apps script，將 Google sheet 中的資料轉換成 JSON 格式，並提供 API 供前端使用。
這是一個問卷調查平台，填寫完問卷後，即可下載獲得一份贈品為Github repository，裡面包含了前端的程式碼以及後端的程式碼，讓使用者可以在本地端運行整個系統。
問卷內容包含了使用者的基本資料、問卷內容以及一些選項，填寫完的資料會被儲存在 Google sheet 中。
包含一個好閱讀的前端介面，讓使用者可以輕鬆地填寫問卷，並且在填寫完後可以下載贈品內容。
